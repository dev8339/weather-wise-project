Weather Wise Pro 🌤️

A high-performance, responsive weather dashboard designed to provide real-time meteorological data with a focus on visual precision and dynamic UI.


🚀 Features

Accurate Local Time: Calculates and displays the exact local time for any searched city worldwide using UTC offset logic.
Dynamic Environments: Automatically toggles between Day/Night modes with CSS-animated celestial bodies (Sun/Moon).
5-Day Forecast: Provides consistent mid-day weather outlooks.


🛠 Tech Stack

Frontend: HTML5, CSS3 (Keyframe Animations, Flexbox, Grid).
Logic: Vanilla JavaScript (ES6+), Asynchronous Fetch API.
Data Source: OpenWeatherMap API.


📂 Project Structure

index.html: Main structure and semantic layout.
style.css: Animations, glassmorphism, and 3D visual effects.
script.js: Logic for API integration, timezone calculations, and DOM manipulation
